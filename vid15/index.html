  1. Internal CSS
Inline:Adding CSS directly to the HTML element using the style attribute.(Prefer not using) Internal: Placing CSS within a style tag in the head section of the HTML document.(For small websites) External: Linking to an external CSS file using the link tag in the head section of the HTML document.(For big websites)
